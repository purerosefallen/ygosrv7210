其他平台的KoishiPro下载

考虑到非Windows操作系统的用户需求，KoishiPro提供一些其他操作系统平台的版本下载。

这些平台的KoishiPro暂不提供自动更新功能。

Linux版本的KoishiPro运行之前需要安装一些运行库。Ubuntu和Debian系统可以运行命令sudo apt install -y libfreetype6-dev libevent-dev libsqlite3-dev libgl1-mesa-dev libglu-dev来安装。RedHat和CentOS系统可以运行命令sudo yum install -y freetype-devel libevent-devel sqlite3-devel libGLU-devel来安装。

若要使用人机对战功能，这些平台需要安装Mono环境。Ubuntu和Debian系统可以运行命令sudo apt install -y mono-complete来安装。RedHat和CentOS系统可以运行命令sudo yum install -y mono-complete来安装。Mac系统可以通过以下链接来下载安装。

自动更新

Windows平台的KoishiPro支持自动更新。

自动更新只需点开update-koishipro目录，运行里面的update.exe即可。

若自动更新出现异常，删除update-koishipro目录下的version.txt并重试即可。
